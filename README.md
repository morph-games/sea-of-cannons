# Sea of Cannons

Play it here: https://morph-games.github.io/sea-of-cannons/dist/

Or on itch: https://deathray.itch.io/sea-of-cannons

## Project Goals

- ✔️ Rendering in pixi.js
- ✖️ 2d planet generation
- ✔️ 2d physics using matter.js lib
- ✔️ fluid/wave simulation
- ✔️ Optional: p2p via peer.js
- ✔️ Gameplay
- ✔️ Submit to GameOff 2025: https://itch.io/jam/game-off-2025

### Non-goals

- WebGL / shaders
- Save/loading
- Complex gameplay
